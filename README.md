# ARM_Projects_TivaC
project 1:
Advanced Calculator:
 takes any expression from the user with multiple operations and prints the result

 Hardware components:
 1- Tiva C EK-TM4C123
 2- keypad
 3- LCD 2x16
 4- potentiometer
 5- breadboard
 6- jumpers

 Software drivers:
 1- TM4C123 registers map https://users.ece.utexas.edu/~valvano/Volume1/tm4c123gh6pm.h
 2- DIO driver
 3- keypad driver
 4- LCD driver
 5- stack driver for the calculator
